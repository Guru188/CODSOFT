# CODSOFT Artificial Intelligence Internship

This repository contains the projects completed as part of the CodSoft Artificial Intelligence Internship.

## Completed Tasks

### Task 1: Rule-Based Chatbot
A simple chatbot that responds to user input using predefined rules and if-else conditions.

### Task 2: Tic-Tac-Toe AI
A Tic-Tac-Toe game where a human player competes against an AI that uses the Minimax algorithm to make optimal decisions.

### Task 4: Recommendation System
A content-based recommendation system that suggests movies to users based on genre similarity using cosine similarity.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## About
These projects demonstrate basic artificial intelligence concepts including rule-based systems, game theory, and recommendation systems.
